## 🔌 **Preparación** 

* Crear un entorno virtual

```bash
virtualenv entorno-rc -p python3
```

* Activar el entorno virtual

```bash
source entorno-rc/bin/activate
```

* Instalar dependencias


```bash
pip install -r requirements.txt
```
